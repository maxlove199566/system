#  練習會員系統

這次主要練習會員註冊  會員登入系統.


## 使用項目

* Bootstrap - 製作簡易的介面
* Node.js - 使用node.js執行本機伺服器
* Express - 使用Express框架
* MongoDB - 資料庫系統
* Mongooes -與MongoDB一起使用連結網頁
* Passport-驗證系統
